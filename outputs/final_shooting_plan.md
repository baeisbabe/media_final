# Final Shooting Plan

This shooting plan is aligned with `outputs/video_production_design.md`, `outputs/image_generation_prompts.md`, and `handoff/06_final_safe_plan.json`.

## Tool Use

| Purpose | Tool | Notes |
|---|---|---|
| Scene images | ChatGPT | Generate 16:9 still images from `outputs/image_generation_prompts.md`; add Chinese text later in Clipchamp. |
| Background music | Suno | Generate clean, upbeat, positive instrumental background music; verify and record terms before submission. |
| Editing | Microsoft Clipchamp | Assemble images/video, subtitles, narration/music, and export `outputs/final_video.mp4`. |

## Production Notes

- Use `outputs/subtitles.srt` as the subtitle base.
- Use Suno track underneath narration/subtitles at a low volume.
- Use Clipchamp to add title cards, subtitle overlays, audio fades, and MP4 export.
- Record every AI-generated image and music file in `outputs/asset_sources.md`.
- Add final MP4 as `outputs/final_video.mp4`.
