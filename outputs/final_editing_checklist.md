# Final Editing Checklist

## Required Output

- [ ] Export final video as `outputs/final_video.mp4`.
- [ ] Confirm total length is between 55 and 65 seconds.
- [ ] Use MP4 format.
- [ ] Use 720p or 1080p resolution.
- [ ] Keep file size reasonable for GitHub upload.

## ChatGPT Images

- [ ] Generate 16:9 scene images using `outputs/image_generation_prompts.md`.
- [ ] Do not let AI generate Chinese title text; add text in Clipchamp.
- [ ] Avoid official Logo, school emblem, brand names, private information, and recognizable unconsented people.
- [ ] Record ChatGPT generation date, prompt reference, output filenames, and terms URL in `outputs/asset_sources.md`.

## Suno Music and Audio

- [ ] Use the Suno-generated instrumental track selected for this project.
- [ ] Record Suno song title, generation date, output filename, and license/terms URL in `outputs/asset_sources.md`.
- [ ] Keep music below narration volume.
- [ ] Add fade in/out at the beginning and end.
- [ ] Do not use copyrighted songs or unverified background music.

## Clipchamp Editing

- [ ] Import ChatGPT images, subtitles, Suno music, and any self-made footage into Clipchamp.
- [ ] Add title cards and Chinese subtitles in Clipchamp.
- [ ] Use simple cuts/fades; avoid distracting transitions.
- [ ] Record Clipchamp as the editing/export tool in `outputs/asset_sources.md`.

## Safety

- [ ] No API key, token, password, cookie or private credential appears.
- [ ] No student ID, grade, private chat or private contact information appears.
- [ ] Recognizable people have consent.
- [ ] Official Logo is used only if permission is confirmed.
- [ ] No unverified ranking, employment rate, admission score, funding, award or partnership claim appears.
