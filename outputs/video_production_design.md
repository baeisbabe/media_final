# Video Production Design

## Project Direction

本影片以「國立臺南大學資訊工程學系 60 秒形象宣傳影片」為主題，採用清楚、正向、可信的校園短片風格。內容重點放在資訊工程學習歷程，而不是誇大招生成果或使用未查證的官方數據。

## Creative Principle

- **可信**：不使用未確認的排名、就業率、錄取分數、競賽成果、外部資金或官方合作資訊。
- **可拍**：優先使用手機或相機可自行拍攝的校園、教室、筆電、白板、討論與 demo 畫面。
- **可讀**：字幕短句化，避免在 60 秒內塞入太多資訊。
- **可交付**：所有素材來源、授權、肖像同意與音樂來源都要能在 `outputs/asset_sources.md` 中追蹤。

| Scene | Time | Production Goal | Visual Design | Narration / Subtitle |
|---|---|---|---|---|
| 01 | 00:00-00:05 | 開場 hook | 校園、鍵盤、筆電、程式碼或快速 B-roll；用 1 個強畫面建立資訊科技感。 | 用資訊技術，把想法變成可以實作的作品。 |
| 02 | 00:05-00:15 | 系所定位 | 系名文字標題、教室或校園外觀；未確認授權前不使用官方 Logo。 | 國立臺南大學資訊工程學系，培養理解問題、設計系統與實作應用的人才。 |
| 03 | 00:15-00:30 | 學習特色 | 程式設計、AI、資料、系統、網路或安全 demo 畫面；螢幕不得出現私人帳號或敏感資料。 | 從程式設計、人工智慧、資料處理到系統與網路，學習不只停在概念，也落實在動手實作。 |
| 04 | 00:30-00:45 | 專題與團隊合作 | 白板討論、小組協作、prototype/demo、簡短展示；不宣稱未查證競賽或研究成果。 | 在專題與團隊合作中，學生練習溝通、分工與展示，把技術轉化成能被理解的成果。 |
| 05 | 00:45-00:55 | 學習氛圍 | 教室、走廊、同儕討論、師生互動；可辨識人物需同意。 | 學習的過程，也包含提問、討論與彼此支援。 |
| 06 | 00:55-01:00 | 收束 CTA | 片尾文字卡，背景可用校園或簡潔底圖。 | 歡迎認識國立臺南大學資訊工程學系。 |

## Visual Style

| Item | Design |
|---|---|
| Color | 乾淨明亮的校園與科技感色調，避免過度炫光或不可讀的深色背景。 |
| Typography | 字幕與標題保持高對比；中文字建議使用剪輯軟體內建清楚字型。 |
| Motion | 簡單 cut、fade 或 slide 即可；不要讓轉場蓋過內容。 |
| Framing | 多用中景與近景，特寫需避開私人資訊與未同意肖像。 |
| Logo | 未確認授權前，以文字「國立臺南大學資訊工程學系」取代官方 Logo。 |

## Narration Design

旁白語氣應保持清楚、穩定、正向，不要像廣告過度誇張。建議錄音時控制在自然語速，必要時刪減句子，讓最終影片仍落在 55 至 65 秒。

| Scene | Narration Role | Delivery Notes |
|---|---|---|
| 01 | 吸引注意 | 短句、稍微有力，進入主題。 |
| 02 | 明確介紹系所 | 語速放慢，讓系名聽清楚。 |
| 03 | 說明學習內容 | 字詞較多，可分兩口氣錄。 |
| 04 | 帶出團隊與專題 | 語氣穩定，搭配討論與 demo 畫面。 |
| 05 | 收斂成學習氛圍 | 語氣溫和，畫面節奏放慢。 |
| 06 | 結尾 CTA | 簡短收尾，保留 0.5 到 1 秒尾音空間。 |

## Music Design

| Item | Recommended Design |
|---|---|
| Music Type | 輕快、乾淨、正向的 royalty-free instrumental。可選 lo-fi tech、light corporate 或 upbeat campus style。 |
| Tempo | 約 90-120 BPM，避免太激烈，以免壓過旁白。 |
| Mood | 可信、年輕、專業，不要使用過度史詩、誇張或情緒太重的配樂。 |
| Intro | 0-5 秒可稍微明亮，配合開場 hook。 |
| Middle | 15-45 秒保持穩定節奏，支撐學習特色與專題畫面。 |
| Ending | 55-60 秒降低音量或做簡單 fade out，讓 CTA 清楚。 |
| License | 必須填入 `outputs/asset_sources.md`，包含音樂名稱、作者、來源 URL、授權條款與查閱日期。 |

## Audio Mixing

- 若有旁白：背景音樂建議約為旁白音量的 20% 到 35%，以字幕與旁白清楚為優先。
- 若只有字幕沒有旁白：音樂可稍高，但仍應避免突兀或過度壓迫。
- 開場與結尾使用 0.5 到 1 秒 fade，避免突然進出。
- 不使用未授權流行歌曲、遊戲音樂、電影配樂或平台不可商用音源。
